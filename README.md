# 🏥 LifeCare - Sistema Web para Clínica de Saúde

## 📌 Descrição

O **LifeCare** é uma aplicação web desenvolvida para representar o site institucional de uma clínica de saúde.
O sistema oferece uma interface moderna, responsiva e intuitiva para apresentar serviços médicos, equipa, contacto e agendamento de consultas.

Este projeto foi desenvolvido com foco em boas práticas de desenvolvimento frontend, usabilidade e design moderno.

---

## 🚀 Tecnologias Utilizadas

* ⚛️ React
* ▲ Next.js
* 🎨 Tailwind CSS
* 🧩 Phosphor Icons
* 🖼️ Next/Image (otimização de imagens)

---

## 🎯 Funcionalidades

* ✅ Página inicial com apresentação da clínica
* ✅ Seção de serviços médicos
* ✅ Apresentação da equipa
* ✅ Depoimentos de pacientes
* ✅ Formulário de contacto
* ✅ Botão de agendamento de consultas
* ✅ Layout responsivo (mobile, tablet e desktop)

## ▶️ Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/seu-usuario/lifecare.git
```

### 2. Entrar na pasta

```bash
cd lifecare
```

### 3. Instalar dependências

```bash
npm install
```

### 4. Executar o projeto

```bash
npm run dev
```

### 5. Acessar no navegador

```
http://localhost:3000
```
